Crie uma aplicação em python chamada crawl4ai-llm-docs que funcionará no terminal e será compatível com Windows e Linux. O objetivo da aplicação é fazer o scraping de sites contendo a documentação de bibliotecas, frameworks, etc e torna-la um único arquivo otimizado para a leitura de llms. A aplicação deverá ser construída em python e deverá utilizar a biblioteca crawl4ai para extração dos dados. O usuário deverá fornecer o caminho de um arquivo .txt contendo as urls contendo a documentação que se quer extrair. Para o modelo de llm utilizado, será utilizado apis compatíveis com a openai. O usuário deverá conseguir configurar modelo, api key e base url para a api. Para testes o arquivo claude-code.txt é um exemplo de arquivo que será fornecido pelo usuário. O documento final deverá abranger todos os dados extraídos de forma otimizada para llms. O output será um arquivo markdown de mesmo nome do arquivo enviado pelo usuário para processamento. Ex: o usuário forneceu o arquivo claude-code.txt, o output com a documentação claude-code.md. A aplicação deverá ser instalável via pipx e poderá ser executada em qualquer pasta. Ela deverá ser interativa, solicitando que o usuário forneça o caminho do arquivo txt com as urls. os dados de configuração e da api utilizada pelo programa deverá ser salvo em um arquivo .json dentro da pasta do usuário, dentro de uma pasta como ".crawl4ai-llm-docs". Ex: ~/.crawl4ai-llm-docs/config.json.
